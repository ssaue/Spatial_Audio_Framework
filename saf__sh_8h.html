Detailed Description

Main header for the Spherical Harmonic Transform and Spherical Array Processing module (SAF_SH_MODULE)

A collection of spherical harmonic related functions. Many of which have been derived from MATLAB libraries by Archontis Politis [1-3] (BSD-3-Clause License).

See also
[1] https://github.com/polarch/Spherical-Harmonic-Transform
[2] https://github.com/polarch/Array-Response-Simulator
[3] https://github.com/polarch/Spherical-Array-Processing
